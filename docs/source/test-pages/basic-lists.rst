Basic lists
===========

Bullet list
-----------

- A bullet list

  + Nested bullet list.
  + Nested item 2.

- Item 2.

  Paragraph 2 of item 2.

  * Nested bullet list.
  * Nested item 2.

    - Third level.
    - Item 2.

  * Nested item 3.

  * This nested list should be compacted by the HTML writer.

    .. _target:

    .. Even if this item contains a target and a comment.
